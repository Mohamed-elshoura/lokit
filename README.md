# Lokit Website

Lokit is a modern fashion e-commerce website designed for online clothing shopping.  
The platform allows users to browse products, search and filter items, view product details, add products to wishlist or cart, place orders, track order status, write reviews, and manage their account information.

## Features

- User authentication: Sign Up, Sign In, Forget Password, Reset Password
- Phone validation / verification flow
- Home page with featured products and categories
- Product search with filters by brand, department, category, size, color, and price range
- Product details page with images, sizes, colors, quantity, reviews, and recommendations
- Wishlist management
- Shopping cart management
- Checkout page with shipping address and payment method
- Order success and failed order pages
- User account section:
  - Orders
  - Wishlist
  - Address
  - Change Password
  - Account Details
- Order details with order status tracking
- Product rating and review system
